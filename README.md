**Project Overview
**

This Power BI dashboard provides a comprehensive analysis of credit card, debit card, and bank transfer transactions. It helps monitor transaction performance, identify high-risk transactions, analyze customer payment behavior, and track branch-wise performance through interactive visualizations.

**Objectives** 

Analyze credit and debit card transactions.

Monitor high-risk and suspicious transactions.

Compare transaction amounts across banks and branches.

Track monthly transaction trends and growth.

Provide interactive insights for business decision-making.
